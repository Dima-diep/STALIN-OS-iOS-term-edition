# Termux-OS-iOS

OS system = Termux

## Installation

For starting, run:
```
$ sh install.sh
```
For changing login (default login is oldlogin), run:
```
$ python3 chlogin.py
```
For changing password (default password is oldpass), run:
```
$ python3 chpass.py
```
For uninstall, run:
```
$ sh uninstall.sh
```
